Controller Package Overview

This index provides links to the detailed documentation for each controller class within the application, facilitating easy navigation and quick access to class functionalities and implementations.

Controller Classes

  • AdminController - Manages administrative functionalities, including user management tasks.
  • AlbumController - Handles album-related operations such as adding, editing, and deleting albums and photos.
  • LoginController - Manages user authentication and directs users to the appropriate dashboard based on their role.
  • PhotoController - Enables detailed photo management within albums, including viewing, tagging, and editing photos.
  • PhotoSearchController - Provides photo search functionalities based on criteria like dates and tags.
  • UserController - Facilitates management of user albums, supporting operations like viewing, adding, and renaming albums.
