Detailed Guide to the PhotoSearchController

The PhotoSearchController enables advanced search capabilities within the application, letting users find photos by date, tags, or a combination of both, enhancing the photo management experience.

Comprehensive Method Descriptions

Understanding the functionalities provided by the PhotoSearchController is crucial for leveraging its full potential in photo searching and organization.

Initialization and User Setup

void start(User user, ArrayList<User> users)

Sets up the search environment based on the current user's context, preparing the system for a customized search experience.

User Interaction and Navigation

void handleLogoutButton(ActionEvent event)

Facilitates secure user logout, ensuring a safe exit point from the search functionality.

void handleBackToAlbumsButton(Event event)

Provides an intuitive navigation path back to the user's album collection, maintaining a seamless application flow.

Search Criteria Management

void handleAddTag(ActionEvent event)

Enhances search accuracy by allowing users to specify tags as part of their search criteria.

void handleRemoveTag(ActionEvent event)

Offers users the flexibility to refine their search parameters by removing previously added tags.

void handleSearchPhotos(ActionEvent event)

Executes the search based on the defined criteria, displaying results that match the user's requirements.

void handleCreateAlbumFromResults()

Empowers users to create a new album from the search results, facilitating organized photo management and quick access to searched photos.
