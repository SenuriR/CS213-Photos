Uses of Package controller

This document outlines the various uses of the classes within the controller package across the application. It serves as a reference to understand the integration points and interactions facilitated by these controllers.

Classes in the controller package

  • AdminController: Manages administrative functions like user management.
  • LoginController: Handles user authentication and navigation to different dashboards based on role.
  • AlbumController: Facilitates album and photo management activities.
  • PhotoController: Provides mechanisms to view and edit individual photos.
  • PhotoSearchController: Implements search functionalities for photos based on various criteria.
  • UserController: Manages user interactions with their albums and photos.

Usage Across the Application

The controllers are utilized in various parts of the application to mediate between the view and model components, ensuring a separation of concerns and facilitating user interactions.

Example: Login Flow

The LoginController is invoked upon application startup. It authenticates user credentials and redirects to the appropriate dashboard. This process involves interactions with models to verify user information and update the view based on the authentication result.
