Controller Package Documentation

This section provides an overview and documentation for the controller package of the application. Each controller is designed to handle specific functionalities within the application, managing the flow between the model and view components.

  • AdminController: Manages administrative functions such as user management.
  • AlbumController: Handles album-related operations including photo management within albums.
  • LoginController: Responsible for handling user logins and directing users to the appropriate dashboard.
  • PhotoController: Manages individual photo operations such as viewing, tagging, and editing.
  • PhotoSearchController: Facilitates searching for photos based on criteria such as tags and dates.
  • UserController: Allows users to manage their photo albums and access various functionalities.
