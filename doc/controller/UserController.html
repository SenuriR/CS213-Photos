UserController: Managing Albums with Ease

The UserController serves as a central hub for album management, enabling users to create, modify, and navigate through their albums with ease.

Detailed Functionalities

The UserController plays a pivotal role in the user interface, directly interacting with the album data and providing a user-friendly experience for album management.

Initialization and Setup

void start(User user, ArrayList<User> users)

Prepares the user interface by loading all albums associated with the current user, setting the stage for album interaction.

User Actions and Interactions

void handleLogOutButton(ActionEvent event)

Ensures a secure logout mechanism, guiding the user back to the login screen.

void handleCancelButton()

Provides users with the option to cancel an ongoing album operation, ensuring a flexible user experience.

void handleConfirmButton()

Facilitates the confirmation of album actions, such as creation or renaming, cementing changes made by the user.

Album Management

void handleAddAlbumButton()

Activates the album addition interface, allowing users to create new albums.

void handleOpenAlbumButton(ActionEvent event)

Opens the selected album, displaying its contents for user interaction and photo management.

void handleDeleteAlbumButton()

Implements album deletion after obtaining user confirmation, ensuring intentional album removal.

void handleRenameButton()

Initiates the album renaming process, enhancing album identification and organization.

void handlesearchPhotosButton(ActionEvent event)

Transitions the user to the photo search interface, expanding the scope of photo management.
