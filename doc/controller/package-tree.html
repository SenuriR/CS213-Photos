Controller Package Class Hierarchy

This document outlines the class hierarchy within the controller package of the photo management application.

Class Hierarchy Overview

All controller classes in the package extend from java.lang.Object, each serving a specific function in the application's MVC architecture.

  • java.lang.Object
    • AdminController - Manages administrative actions such as user management.
    • AlbumController - Manages operations on albums, including photo manipulation.
    • LoginController - Handles user login and initial setup procedures.
    • PhotoController - Facilitates viewing and tagging individual photos.
    • PhotoSearchController - Implements search functionality across the application.
    • UserController - Manages user interactions with their albums and photos.

For more detailed information on each controller, please refer to the specific documentation page linked above.
