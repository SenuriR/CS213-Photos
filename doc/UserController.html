UserController Class

The UserController class facilitates album management within the application, offering functionalities like viewing, renaming, and deleting albums.

Functional Overview

This controller is integral for the album management aspect of the application, allowing users to interact with their photo albums directly. It handles album creation, deletion, and renaming, besides providing a gateway to album content and search functionalities.

Method Summary

  • void start(User user, ArrayList<User> users) - Initializes the controller with the current user and their albums.
  • void handleLogOutButton(ActionEvent event) - Handles logout action to return to the login screen.
  • void handleCancelButton() - Confirms cancellation of the current operation.
  • void handleConfirmButton() - Confirms the addition of a new album or renaming of an existing one.
  • void handleAddAlbumButton() - Prepares the interface for adding a new album.
  • void handleOpenAlbumButton(ActionEvent event) - Opens the selected album to view its content.
  • void handleDeleteAlbumButton() - Deletes the selected album after confirmation.
  • void handleRenameButton() - Prepares the interface for renaming the selected album.
  • void handlesearchPhotosButton(ActionEvent event) - Navigates to the photo search interface.

Usage Example

The UserController is primarily used in the main user dashboard, where it manages the user's interaction with their albums. It orchestrates the UI response to user actions such as album creation, deletion, and renaming, enhancing the application's usability.
