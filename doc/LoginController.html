LoginController Class

Responsible for managing the login process, including verifying user credentials and navigating to the appropriate dashboard based on the user role (admin, stock user, or regular user).

Class Overview

The LoginController class processes input from the login view, authenticates users, and directs them to either the admin dashboard, stock dashboard, or user dashboard. It also handles first-time setup by creating necessary data files and stock user accounts if they don't exist.

Method Details

  • public void start(Stage stage)

    Initializes the controller. This method is a placeholder and may contain setup logic as needed.

  • public void handleLoginButton(ActionEvent event)

    Handles the login process. It reads the username from the input field, checks for the existence of the data file, and validates the user. Depending on the user type, it loads the corresponding dashboard.

Usage

This controller is bound to the login view FXML file and is invoked when the application starts and presents the login screen to the user. The handleLoginButton method is called when the user clicks the login button.
