AlbumController Class

The AlbumController class is responsible for managing the album's photos including operations like adding, deleting, moving, and copying photos within albums. It interacts with the album view to reflect these changes in real-time.

Method Summary

  • void start(ArrayList<User> users, User user, Album selectedAlbum) - Initializes the controller with users, the current user, and the selected album.
  • void handleAddPhotoButton() - Opens a file chooser to add a new photo to the album.
  • void handleAlterPhotoButton(ActionEvent event) - Prepares the interface for photo alteration operations based on the event source.
  • void handleCancelButton() - Cancels the current operation and resets the interface.
  • void handleConfirmButton() - Confirms the addition, deletion, or alteration of a photo and saves changes.
  • void handleDeletePhotoButton() - Deletes the selected photo from the album.
  • void handleOpenPhotoButton(ActionEvent event) - Opens the selected photo for viewing.
  • void handleBackButton(ActionEvent event) - Returns to the user dashboard view.
  • void handleLogoutButton(ActionEvent event) - Logs out the current user and returns to the login screen.
