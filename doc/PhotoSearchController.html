PhotoSearchController Class

The PhotoSearchController class facilitates the search functionality within the application, allowing users to search for photos based on date ranges and tags.

Functional Overview

This controller is responsible for handling user inputs for photo searches, including date range selection and tag filtering. It displays the search results in a list view, enabling users to view or create an album from these results.

Method Summary

  • void start(User user, ArrayList<User> users) - Initializes the controller with the current user and a list of all users.
  • void handleLogoutButton(ActionEvent event) - Logs out the current user and returns to the login screen.
  • void handleBackToAlbumsButton(Event event) - Navigates back to the user dashboard, displaying all albums.
  • void handleAddTag(ActionEvent event) - Adds a tag to the search criteria list.
  • void handleRemoveTag(ActionEvent event) - Removes a selected tag from the search criteria list.
  • void handleSearchPhotos(ActionEvent event) - Performs the search based on the specified criteria and displays results.
  • void handleCreateAlbumFromResults() - Creates a new album from the search results.

Usage Example

The PhotoSearchController is used in scenarios where users wish to find photos within their collection based on specific criteria. Upon entering search parameters and executing a search, the user is presented with matching photos, which can then be compiled into a new album.
